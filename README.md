## Projects

I use GitHub as a portfolio to showcase finished or presentable projects.
Other works are stored in alternative forges and mirrored/migrated based on progress.

### GDPR Enforcement Pipeline

A semi-automated pipeline that fetches enforcement decisions from Data Protection
Authorities, standardizes them into a common schema, and uses a local LLM with
Tesseract OCR to process supporting documents. Results are visualized with D3.js.

### My Qubes OS Journey

An ever-evolving documentation project tracking my experience with Qubes OS —
covering installation, configuration, and automation. Built as a straight-forward
guide for anyone interested in getting started with Qubes.

### Arch Linux Installer & Dotfiles

A script that installs Arch Linux with my own configuration and dotfiles. Built
with guidance from *Mouseless Dev*, which I highly recommend.
